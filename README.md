## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash

# install postgresql
$ docker compose up -d

# dependencies
$ npm install

# prisma migrate
$ npm run migrate:dev
```

## seed data

manually run the sql file in prisma/migrations/20241206142005_init_schema_for_jenosize_e_commerce/seed.sql

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
